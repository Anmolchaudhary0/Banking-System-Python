# Banking-System-Python
A Python-based Bank Management System featuring both Console and Streamlit interfaces. The application supports account creation, authentication, deposits, withdrawals, account management, and secure data storage using JSON. Built using Object-Oriented Programming (OOP) principles to demonstrate banking operations and financial management.
# 🏦 Bank Management System

A Python-based Bank Management System developed with both **Console** and **Streamlit Web Application** interfaces. This project simulates basic banking operations such as account creation, deposits, withdrawals, account management, and user authentication using a JSON-based database.

## 🌐 Live Demo

Streamlit Application: https://banking-system-python-xrqzaxjuxzdk3flebjkgd4.streamlit.app/

## 📌 Features

### Console Version
- Create New Account
- Deposit Money
- Withdraw Money
- View Account Details
- Update Account Information
- Delete Account
- JSON File Storage

### Streamlit Web Version
- User-Friendly Web Interface
- Create New Account
- Deposit Money
- Withdraw Money
- View Account Details
- Delete Account
- Secure Login Authentication
- JSON File Storage

## 🛠️ Technologies Used

- Python
- Streamlit
- JSON
- Object-Oriented Programming (OOP)
- File Handling
